This section is reserved for structured pages on the main hardware categories used in industrial digitalization projects.

## Examples of coverage

- Remote I/O, gateways, HMIs, and edge computers
- Industrial switches, routers, and protocol converters
- Sensors, panels, and supporting infrastructure hardware

The emphasis is on what each device class does, where it fits, and how it should be evaluated.
