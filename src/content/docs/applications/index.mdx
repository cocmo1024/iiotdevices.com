# Applications

Applications are the cleanest place to start industrial device research. Environment, asset behavior, maintenance reality, and data goals usually determine the right hardware class long before brand preference becomes useful.

<div class="signal-strip">

## Decision flow

<Steps>
	1. Define the site environment, asset behavior, and data objective.
	2. Identify what must be sensed, controlled, stored locally, or sent upstream.
	3. Narrow the likely device families and interoperability requirements.
	4. Compare vendors only after the application boundaries are stable.
</Steps>
